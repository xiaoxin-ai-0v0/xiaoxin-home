# Java 学习路线分享

这篇文章记录我的 Java 学习方法，重点是按阶段构建能力，而不是只看零散知识点。

## 阶段一：基础能力

- 语法、面向对象、异常处理
- 常见数据结构与集合框架
- I/O 与并发基础

## 阶段二：工程化能力

- Maven/Gradle 依赖管理
- 单元测试与接口测试
- 日志、配置、部署流程

## 阶段三：业务开发能力

- Spring Boot 开发 REST API
- MySQL 设计与索引优化
- 缓存与性能监控

## 我的建议

先把一个小项目做完整：开发、测试、部署、复盘。完整闭环比刷题更能提升真实交付能力。
