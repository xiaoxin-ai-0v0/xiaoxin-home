# AI 时代，开发者如何转型

AI 不会替代所有开发者，但会重塑开发者的工作方式。关键是把 AI 变成你的生产力工具。

## 先补哪部分

1. Python 基础与数据处理
2. LLM API 调用与提示词工程
3. 向量检索、工作流编排和部署

## 如何落地到项目

- 从一个真实问题开始（如知识库问答）
- 先做 MVP，再迭代质量
- 用日志和指标追踪效果

## 结语

转型不是推倒重来，而是把已有工程经验迁移到新的技术栈中。
