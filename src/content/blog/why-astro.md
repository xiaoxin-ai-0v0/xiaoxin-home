# 为什么我从 React 转向了 Astro？

大家好，我是小新。今天我想分享一下为什么我从 React 转到了 Astro 这个框架。

## 背景

作为一个全栈开发者，我一直在使用 React 来构建各种项目。但是随着项目规模的增大，我开始遇到了性能问题。

## Astro 的优势

### 1. 岛屿架构

Astro 采用了一种叫做"岛屿架构"的技术。这意味着页面中的静态内容不会被 JavaScript 化，只有需要交互的部分才会被水合。

### 2. 零 JS 默认

默认情况下，Astro 不会向浏览器发送任何 JavaScript。这对于内容型网站来说简直完美。

### 3. 更好的性能

由于减少了 JavaScript 的加载，页面的加载速度大幅提升。

## 总结

如果你正在构建一个内容为主的网站，我强烈推荐你试试 Astro。它真的改变了我的开发体验！
