# 为什么我从 React 转向 Astro

在个人站点场景下，Astro 的静态优先策略更适合内容型网站。

## 核心原因

### 页面更轻

默认输出静态 HTML，首屏加载更快。

### 内容管理更清晰

配合 Content Collections，文章和项目数据结构更稳定。

### 仍可按需上岛

需要交互时再引入 React 组件，不会全站水合。

## 适用场景

- 个人博客
- 作品集站点
- 文档型站点
