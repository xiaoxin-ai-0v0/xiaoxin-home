# 项目概览

这个项目用于构建个人品牌展示站，目标是“内容可运营、结构可扩展”。

## 功能

- 博客内容管理
- 项目展示与详情页
- 路由化导航与 SEO

## 技术栈

- Astro
- Tailwind CSS
