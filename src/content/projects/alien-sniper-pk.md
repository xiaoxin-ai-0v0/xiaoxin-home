# 项目概览

这是一个围绕“好友 PK”做的小游戏 demo。核心目标是验证对战循环和玩法反馈。

## 功能

- 本地双人对战
- 外星人扰动机制
- 击杀计分与 60 秒结算

## 技术栈

- 原生 JavaScript
- HTML5 Canvas
